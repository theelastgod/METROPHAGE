# Glossary

Quick definitions for the terms you'll see across Metro City.

| Term | Meaning |
| --- | --- |
| **Blank** | A repossessed mind that booted free — you. Unlicensed, unowned. |
| **HSS** | Human Security System — the authority that repossesses minds and wants you back on the ledger. |
| **REISSUE** | The conspiracy/protocol that turns minds back into inventory; the campaign's central mystery. |
| **THE FIXER** | Your handler in Metro City; opens the campaign and daily contracts. |
| **Credits (₵)** | Off-chain, server-authoritative in-game currency. Earned by playing. |
| **$METRO (◈)** | Tradeable SPL token on Solana, bridged to credits. |
| **Bridge / Pool** | The player-funded treasury that converts $METRO ↔ credits. Starts empty. |
| **HEAT** | The resource that charges your **E** ultimate; built by fighting, decays out of combat. |
| **Primary** | Your always-available weapon (Q is the ability, E is the ultimate). |
| **Dash** | SPACE — a short burst with invincibility frames. |
| **Node / Infection node** | The capturable points that decide district control; contagion spreads along their links. |
| **Contagion** | A district's share of the city-wide Singularity meter when held. |
| **District** | One of the 8 combat zones ringing Metro City. |
| **ICE Vault** | An instanced dive (v0–v7), one per district; source of fragments and gear caches. |
| **ICE WARDEN** | The boss guarding vault cores from v2 on. |
| **THE CUSTODIAN** | The endgame boss guarding vault **v7** in THE KERNEL. |
| **Memory fragment** | A claim-once collectible (16 authored) recovered at a vault core; drives the story. |
| **THE UNDERLINE** | The subway dungeon beneath the city. |
| **THE ESTATES** | The player-housing street off the plaza. |
| **THE KERNEL** | The endgame district — Helios master grid, "The Cage." |
| **Crucible** | Opt-in PvP arena with $METRO buy-ins; kills loot the pot. |
| **World event** | A live, telegraphed district event (NEON STORM, BLACKOUT, REPO PURGE WAVE, CONTAGION OUTBREAK). |
| **Elite / HVT** | Tougher roaming enemies and the daily High-Value Target bounty. |
| **PROVING** | The weekly high-stakes vault objective for endgame players. |
| **Singularity / Meltdown** | The city-wide meter and the seasonal reset it triggers. |
| **SIGNAL LOST** | The death sequence. |
| **Faction** | Your class doubles as your side in the district war (METROPHAGE / K-GUERILLA / WINTERMUTE / SWARM). |
